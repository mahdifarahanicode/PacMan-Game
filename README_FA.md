## معرفی

این پروژه بازسازی بازی کلاسیک **Pac-Man** با استفاده از **Python** و **Pygame** است.

در کنار بازسازی نسخه کلاسیک، امکانات جدیدی نیز به بازی اضافه شده است؛ از جمله هوش مصنوعی چندمرحله‌ای، پاور پلت، ذخیره رکورد، سیستم مراحل، ظاهر پیکسلی و افکت‌های صوتی.

هدف از توسعه این پروژه، تمرین مباحث زیر بوده است:

- برنامه‌نویسی شی‌گرا
- توسعه بازی
- هوش مصنوعی
- الگوریتم BFS
- طراحی ساختار پروژه

---

## امکانات

- ۶ مرحله با افزایش سختی
- سه نوع هوش مصنوعی برای روح‌ها
- پاور پلت
- تم رنگی متفاوت برای هر مرحله
- ذخیره رکورد
- منوی Pause
- صداگذاری
- اسپرایت‌های پیکسلی
- حرکت نرم روی گرید
